## 🚀 快速开始指南

### 新项目推荐流程：

1. **探索想法**（可选）
   ```
   /bmad-bmm-create-product-brief
   ```

2. **创建PRD**（必需）
   ```
   /bmad-bmm-create-prd
   ```

3. **创建架构**（必需）
   ```
   /bmad-bmm-create-architecture
   ```

4. **创建史诗和故事**（必需）
   ```
   /bmad-bmm-create-epics-and-stories
   ```

5. **检查实施就绪**（必需）
   ```
   /bmad-bmm-check-implementation-readiness
   ```

6. **开始冲刺计划**（必需）
   ```
   /bmad-bmm-sprint-planning
   ```

### 现有项目快速开始：

```
/bmad-bmm-generate-project-context
```

这将扫描你的代码库并生成 AI 优化的项目上下文文件。

## ⚠️ 重要说明

**当前状态：**
- ✅ Agents 会用中文与你交流
- ✅ 生成的文档使用中文
- ⚠️ 斜杠命令仍需使用英文版本（如 `/bmad-bmm-create-prd`）
- 📝 中文命令映射文件已创建在 `_bmad/_config/bmad-help-zh.csv`

**为什么命令仍是英文？**
BMAD 的命令系统基于 skill 注册机制，目前不支持动态命令别名。使用中文配置后，所有交互和输出都是中文，但命令本身需要使用英文。
