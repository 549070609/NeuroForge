## 📖 重要说明

### ✅ 已汉化：
- Agents 对话语言 → 中文
- 文档输出语言 → 中文
- 提示和指导 → 中文

### ⚠️ 仍为英文：
- 斜杠命令本身（如 `/bmad-bmm-create-prd`）
- 系统内部文件路径

**原因：** BMAD 的命令注册系统基于英文，但不影响中文使用体验。
