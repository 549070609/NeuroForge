## 关键模块映射 (v2.x → v3.0)

| v2.x 路径 | v3.0 路径 | 说明 |
|-----------|-----------|------|
| `core/events.py` | 废弃 | 兼容层，移除 |
| `core/failover.py` | 废弃 | 兼容层，移除 |
| `plugins/integration/events/` | 保持 | EventBus，遥测数据源 |
| `plugins/middleware/failover/` | 保持 | ProviderPool，遥测数据源 |
| `config/` | `foundation/config/` | 迁移 |
| `commands/` | `capabilities/commands/` | 迁移 |
| `skills/` | `capabilities/skills/` | 迁移 |
| `tools/` | `capabilities/tools/` | 迁移 |
