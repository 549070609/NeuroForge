# 编辑文件

智能编辑文件，支持查找和替换。

## 参数说明

用户输入格式：`/edit [文件名或模式] [修改描述]`

示例：
- `/edit main.py 修复 bug`
- `/edit **/*.py 重命名变量`
- `/edit config.yaml 更新配置`

---

## 编辑任务

请执行以下操作：

### 1. **定位文件**

根据用户输入：
- 如果指定了文件名，直接定位
- 如果是模式，使用 glob 搜索
- 如果未指定，列出最近修改的文件

### 2. **读取当前内容**

显示文件的相关部分：
- 显示完整文件（小文件）
- 或显示上下文（大文件）

### 3. **理解修改意图**

分析用户的修改描述：
- 明确修改目标
- 识别影响范围
- 确认修改方式

### 4. **执行修改**

使用 edit 工具：
- 精确定位要修改的代码
- 保持代码风格一致
- 不破坏现有功能

### 5. **验证修改**

- 显示修改前后的对比
- 确认修改符合预期
- 检查是否有遗漏

---

## 编辑原则

1. **最小化修改** - 只修改必要的部分
2. **保持一致性** - 遵循现有代码风格
3. **向后兼容** - 不破坏现有接口
4. **添加注释** - 对复杂修改添加说明

---

请等待用户提供文件名和修改描述，然后执行编辑操作。
