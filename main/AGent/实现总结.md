## 🤖 三个主 Agent

### 1. 构思专家 (novel-ideation)

- **格式**：YAML
- **职责**：世界观构建、人物设定、主题确定
- **温度**：0.9（高创意）
- **成本**：MODERATE
- **依赖**：无（基础层）
- **特点**：
  - 使用 YAML 格式，可读性最好
  - 高温度激发创意
  - 详细的系统提示词
  - 委托触发器配置

### 2. 大纲专家 (novel-outline)

- **格式**：JSON
- **职责**：章节规划、情节设计、节奏控制
- **温度**：0.7（结构化）
- **成本**：MODERATE
- **依赖**：novel-ideation
- **特点**：
  - 使用 JSON 格式，易于程序生成
  - 中等温度保持结构
  - 依赖构思专家的输出
  - 支持多种叙事框架

### 3. 写手 (novel-writer)

- **格式**：Python
- **职责**：章节撰写、场景描写、对话创作
- **温度**：0.85（创意写作）
- **成本**：EXPENSIVE
- **依赖**：novel-outline
- **特点**：
  - 使用 Python 格式，最灵活
  - 大 token 限制（8192）
  - 持久化会话
  - 跨章节记忆

## ✅ 核心功能展示

### 1. 多格式支持

| Agent | 格式 | 文件大小 | 优势 |
|-------|------|---------|------|
| 构思专家 | YAML | 3.5 KB | 可读性好，适合手写 |
| 大纲专家 | JSON | 2.8 KB | 易于程序生成和解析 |
| 写手 | Python | 5.2 KB | 最灵活，可包含复杂逻辑 |

### 2. 依赖管理

```python
# 依赖定义
novel-ideation:
  requires: []

novel-outline:
  requires: ["novel-ideation"]

novel-writer:
  requires: ["novel-outline"]
  optional_requires: ["novel-ideation"]

# 自动解析加载顺序
loader.resolve_dependencies(["novel-writer", "novel-ideation", "novel-outline"])
# 返回: ["novel-ideation", "novel-outline", "novel-writer"]
```

### 3. AgentBuilder 流畅 API

```python
schema = (AgentBuilder()
    .with_name("novel-ideation")
    .with_temperature(0.9)
    .add_tools(["read", "write"])
    .with_prompt("你是一位构思专家...")
    .use_when(["构思", "创意"])
    .build_and_register())
```

### 4. AgentLoader 多格式加载

```python
# 从不同格式加载
yaml_agent = loader.load("agents/ideation.yaml")
json_agent = loader.load("agents/outline.json")
python_agent = loader.load("agents/writer.py")

# 加载整个目录
all_agents = loader.load_directory("agents/")
```

### 5. AgentFactory 实例管理

```python
# 创建实例
ideation = factory.create_from_name("novel-ideation")
outline = factory.create_from_name("novel-outline")
writer = factory.create_from_name("novel-writer")
```

### 6. Registry 智能发现

```python
# 按标签查询
novel_agents = registry.find_by_tags(["novel"])
# 返回: [novel-ideation, novel-outline, novel-writer]

# 按能力查询
write_agents = registry.find_by_capability("write")
# 返回: [novel-ideation, novel-outline, novel-writer]

# 智能匹配
best = registry.find_best_for_task("我需要构思一个科幻小说")
# 返回: novel-ideation
```

```python

# 创建 Factory 和 Loader
factory = AgentFactory(provider_factory, tool_registry)
loader = AgentLoader(factory)

# 加载 Agent
loader.load_directory("agents/")

# 创建实例
ideation = factory.create_from_name("novel-ideation")

result = ideation.run("构思一个科幻小说...")
```
