# 写小说系统增强方案

基于当前的基础系统，集成更多高级能力。

## 🚀 增强目标

### 1. 集成长记忆能力

**目标**：让 Agent 能够记住跨项目的经验和知识

**实现**：
- 集成 `E:\localproject\Agent Learn\main\Long-memory` 的向量存储
- Agent 可以查询历史小说的创作经验
- 记住成功的情节模式、人物原型、写作技巧

**应用场景**：
```
构思专家: "我之前创作过3本科幻小说，总结出了5种常见的世界观模式..."
大纲专家: "根据历史数据，悬疑小说的最佳章节长度是3000-5000字..."
写手: "参考之前成功的对话模式，这个场景应该这样写..."
```

### 2. 多编辑讨论系统

**新增 Agent**：
- **编辑A（保守派）**：注重传统、稳妥
- **编辑B（创新派）**：追求新颖、突破
- **主编（协调者）**：平衡双方意见

**工作流**：
```
写手完成初稿
    ↓
编辑A评审 + 编辑B评审
    ↓
产生不同意见
    ↓
主编协调讨论
    ↓
综合意见反馈给写手
    ↓
写手修改
```

### 3. 迭代改进循环

**实现**：
- 质量评分系统
- 自动识别薄弱环节
- 针对性改进建议
- 多轮迭代直到达标

    2. 识别分歧点和共同点

#### 2.2 协作工作流

```python
# workflow_with_editors.py

from pyagentforge.building import AgentLoader, AgentFactory

# 1. 加载所有 Agent
loader.load_directory("agents/")

# 2. 创建实例
writer = factory.create_from_name("novel-writer")
editor_a = factory.create_from_name("editor-conservative")
editor_b = factory.create_from_name("editor-innovative")
chief = factory.create_from_name("editor-chief")

# 3. 写手完成初稿
draft = writer.run("撰写第1章")

# 4. 两位编辑并行评审
review_a = editor_a.run(f"评审：\n{draft}")
review_b = editor_b.run(f"评审：\n{draft}")

# 5. 主编协调
decision = chief.run(
    f"初稿：\n{draft}\n\n"
    f"编辑A意见：\n{review_a}\n\n"
    f"编辑B意见：\n{review_b}\n\n"
    "请综合意见并给出修改建议"
)

# 6. 写手修改
revision = writer.run(
    f"根据主编建议修改第1章：\n{decision}"
)
```

### Phase 3: 质量评分与迭代

#### 3.1 评分系统

```python
# quality_scorer.py

class NovelQualityScorer:
    """小说质量评分器"""

    def score(self, chapter: str) -> dict:
        """
        评分维度：
        - 情节连贯性 (0-10)
        - 人物塑造 (0-10)
        - 对话自然度 (0-10)
        - 场景描写 (0-10)
        - 文笔优美度 (0-10)
        - 整体可读性 (0-10)
        """
        return {
            "plot_coherence": 8.5,
            "character_depth": 7.2,
            "dialogue_natural": 8.0,
            "scene_description": 9.0,
            "writing_style": 8.5,
            "overall_readability": 8.3,
            "average": 8.25,
        }

    def identify_weaknesses(self, scores: dict) -> list[str]:
        """识别薄弱环节"""
        weaknesses = []
        for dimension, score in scores.items():
            if score < 8.0 and dimension != "average":
                weaknesses.append(dimension)
        return weaknesses
```

#### 3.2 迭代改进

```python
# iterative_improvement.py

async def improve_chapter(
    chapter_num: int,
    target_score: float = 8.5,
    max_iterations: int = 3
):
    """迭代改进章节"""

    writer = factory.create_from_name("novel-writer")
    scorer = NovelQualityScorer()

    # 初稿
    chapter = writer.run(f"撰写第{chapter_num}章")
    score = scorer.score(chapter)

    iteration = 1
    while score["average"] < target_score and iteration <= max_iterations:
        # 识别薄弱环节
        weaknesses = scorer.identify_weaknesses(score)

        # 针对性改进
        improvement_prompt = f"""
        当前第{chapter_num}章的评分：
        {json.dumps(score, indent=2)}

        薄弱环节：{', '.join(weaknesses)}

        请针对性改进这些方面：
        {chapter}
        """

        chapter = writer.run(improvement_prompt)
        score = scorer.score(chapter)
        iteration += 1

    return chapter, score
```

### 2. 并行 Agent

```python
import asyncio

async def parallel_review(chapter: str):
    """并行评审"""
    tasks = [
        editor_a.run(f"评审：{chapter}"),
        editor_b.run(f"评审：{chapter}"),
    ]
    reviews = await asyncio.gather(*tasks)
    return reviews
```

from langgraph.graph import StateGraph
